## 1. Обзор продукта

**shorterLINK** — внутренний инструмент для laptop.guru, который автоматизирует процесс обработки клиентских заявок и отправки персонализированных email с видеообзорами товаров.

**Одно предложение:** Клиент оставляет заявку → сотрудник подбирает видеообзор → система генерирует красивый лендинг и email → клиент получает письмо со ссылкой на видео и кнопкой «Купить».

---

## 2. Проблема

### 2.1 Текущий процесс (AS-IS)

| Шаг | Действие | Время | Боль |
|-----|----------|-------|------|
| 1 | Клиент заполняет форму на laptop.guru | — | — |
| 2 | Письмо приходит на почту | — | — |
| 3 | Сотрудник читает письмо | 2 мин | Ручной поиск ссылки на товар |
| 4 | Сотрудник ищет видеообзор на YouTube | 5 мин | Нет библиотеки видео, каждый раз заново |
| 5 | Сотрудник вручную формирует ответ | 10 мин | Нет шаблонов, некрасивые текстовые письма |
| 6 | Сотрудник отправляет email | 2 мин | Нет трекинга, не знаешь открыл ли клиент |
| **Итого** | | **~20 мин/заявку** | **Медленно, некрасиво, нет аналитики** |

### 2.2 Целевой процесс (TO-BE)

| Шаг | Действие | Время |
|-----|----------|-------|
| 1 | Система подтягивает заявки с почты автоматически | 0 мин |
| 2 | Сотрудник выбирает видео из готовой библиотеки | 1 мин |
| 3 | Система генерирует лендинг + email | 0 мин |
| 4 | Сотрудник нажимает «Отправить» | 10 сек |
| **Итого** | | **~1-2 мин/заявку** |

**Экономия: ~18 минут на каждую заявку (90% сокращение времени).**

| Атрибут | Описание |

## 7. Scope

### 7.1 В скоупе MVP (v1.0)

- Регистрация/авторизация с инвайт-кодом
- IMAP-подключение к Hostinger, парсинг заявок
- Добавление YouTube-видео по ссылке с автоматическим подтягиванием метаданных
- Генерация лендинга с iframe-видео и кнопкой «Купить»
- Генерация коротких ссылок (nanoid)
- HTML-email шаблон через React Email
- Ручная отправка email (выбор заявки + видео → отправка)
- Базовая статистика (просмотры, клики)
- История отправленных писем
- Ручное редактирование данных заявки (когда парсер ошибается)
- Confirm dialog перед отправкой
- Архивация нерелевантных заявок
- Копирование ссылок в буфер
- Toast-уведомления
- Docker-деплой на VPS с Caddy (SSL)

### 7.2 НЕ в скоупе MVP

| Фича | Причина | Когда |
|-------|---------|-------|
| Автоматическая отправка email | Сначала нужно стабилизировать ручной процесс | v2.0 |
| Несколько шаблонов email | Один шаблон достаточен для старта | v2.0 |
| Open rate / delivery tracking | Требует webhook-интеграции с email-провайдером | v2.0 |
| A/B тестирование лендингов | Не хватает данных для статистической значимости на старте | v3.0 |
| Интеграция с CRM | Нет CRM на текущий момент | v3.0 |
| Telegram-бот уведомления | Nice-to-have, не критично для MVP | v2.0 |
| Мульти-язычность | Текущая аудитория — один регион | v3.0 |

## 9. Зависимости и риски

### 9.1 Внешние зависимости

| Зависимость | Тип | Влияние | Митигация |
|-------------|-----|---------|-----------|
| Hostinger IMAP/SMTP | Сервис | Без почты не работает парсинг и отправка | Retry-логика, fallback-уведомления в UI |
| YouTube Data API v3 | Сервис | Без API нельзя подтягивать метаданные видео | Квота 10,000 ед/день (достаточно). Fallback: oEmbed (без API key) |
| YouTube iframe | Сервис | Видео на лендинге не загрузится если YouTube недоступен | Показывать превью-картинку как fallback |
| laptop.guru (форма) | Формат | Если формат писем изменится — парсер сломается | Regex-парсер вынесен в отдельный модуль, легко обновить |

### 9.2 Риски

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| IMAP-таймауты на Hostinger | Средняя | Заявки не подтягиваются | Таймаут 30 сек, retry, ошибка в UI |
| YouTube API квота исчерпана | Низкая | Нельзя добавлять видео | oEmbed fallback, кеширование метаданных |
| Письма попадают в спам | Средняя | Клиент не получает email | SPF/DKIM/DMARC настройка на домене, тестирование |
| Изменение формата заявок laptop.guru | Низкая | Парсер не извлекает данные | Модульный парсер, raw-fallback |
| VPS недоступен | Низкая | Весь сервис недоступен | Docker restart-policy, мониторинг |

## 11. Глоссарий

| Термин | Определение |
|--------|------------|
| **Заявка** | Письмо от клиента, пришедшее на почту после заполнения формы на laptop.guru |
| **Видеотека** | Внутренняя библиотека YouTube-видео с обзорами товаров |
| **Лендинг** | Одностраничный сайт с видеообзором и кнопкой «Купить», генерируемый системой |
| **Короткая ссылка** | URL вида `/r/[code]`, который редиректит на лендинг |
| **Инвайт-код** | Секретный код для регистрации, защищает от посторонних |
| **Парсер** | Модуль, извлекающий из тела email: ссылку на товар, данные клиента |
